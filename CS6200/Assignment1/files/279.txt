<p><b>Porter</b> (\p(o)-rter\) is a common <a href="/wiki/English_language" title="English language">English</a> <a href="/wiki/Surname" title="Surname">surname</a> and also a <a href="/wiki/Given_name" title="Given name">given name</a>. The name originates as an <a href="/wiki/Old_French" title="Old French">Old French</a> occupational name, <i>portier</i> (gatekeeper; doorkeeper), or <i>porteour</i> ("to carry"). Its earliest public record is 1086 at <a href="/wiki/Winchester_Castle" title="Winchester Castle">Winchester Castle</a>.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> With transferred use, Porter also became a <a href="/wiki/Masculine" class="mw-redirect" title="Masculine">masculine</a> given name with varied popularity. According to the U.S. <a href="/wiki/Social_Security_Administration" title="Social Security Administration">Social Security Administration</a>, Porter ranked #433 in 1907, declined to #1002 in 1944, then rebounded to #476 in 2006.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup><sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup>
