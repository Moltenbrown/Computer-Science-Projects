	<h1 id="firstHeading" class="firstHeading" lang="en">Mason (surname)</h1>
	
	<div id="bodyContent" class="mw-body-content">
		<div id="siteSub" class="noprint">From Wikipedia, the free encyclopedia</div>
		<div id="contentSub"></div>
		
		
		
		<div id="jump-to-nav"></div>
		<a class="mw-jump-link" href="#mw-head">Jump to navigation</a>
		<a class="mw-jump-link" href="#p-search">Jump to search</a>
		<div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><p><b>Mason</b> is an <a href="/wiki/Italian_name" title="Italian name">Italian</a>, <a href="/wiki/French_name" title="French name">French</a>, <a href="/wiki/Irish_(name)" title="Irish (name)">Irish</a> or <a href="/wiki/English_name" title="English name">English</a> <a href="/wiki/Surname" title="Surname">surname</a> that refers to someone who did <a href="/wiki/Stonemasonry" title="Stonemasonry">stonemasonry</a> work, or it derives from the <a href="/wiki/Given_name" title="Given name">given name</a> "Maso", which is the short form of the <a href="/wiki/Personal_name" title="Personal name">personal name</a> "<a href="/wiki/Tommaso" title="Tommaso">Tommaso</a>". The name "Tommaso", or "<a href="/wiki/Thomas_(name)" title="Thomas (name)">Thomas</a>", is itself derived from the <a href="/wiki/Aramaic_language" class="mw-redirect" title="Aramaic language">Aramaic</a> <i>toma</i> ("<a href="/wiki/Twin" title="Twin">twin</a>").<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup><sup id="cite_ref-HouseOfNames_2-0" class="reference"><a href="#cite_note-HouseOfNames-2">&#91;2&#93;</a></sup>
</p><p>In Italy, the name was first recorded in <a href="/wiki/Piedmont" title="Piedmont">Piedmont</a>, in France in <a href="/wiki/Provence" title="Provence">Provence</a> and in <a href="/wiki/England" title="England">England</a>, on the <a href="/wiki/Isle_of_Thanet" title="Isle of Thanet">Isle of Thanet</a>.
</p><p>Variants of the surname include <a href="/wiki/Masella_(disambiguation)" class="mw-redirect" title="Masella (disambiguation)">Masella</a>, <a href="/wiki/Maselli" title="Maselli">Maselli</a>, Masellis, <a href="/wiki/Masetti" title="Masetti">Masetti</a>, <a href="/wiki/Masetto_(disambiguation)" class="mw-redirect mw-disambig" title="Masetto (disambiguation)">Masetto</a>, <a href="/wiki/Masi_(surname)" title="Masi (surname)">Masi</a>, <a href="/wiki/Masiello" title="Masiello">Masiello</a>, <a href="/w/index.php?title=Masillo&amp;action=edit&amp;redlink=1" class="new" title="Masillo (page does not exist)">Masillo</a>, <a href="/wiki/Masin_(disambiguation)" class="mw-redirect mw-disambig" title="Masin (disambiguation)">Masin</a>, <a href="/wiki/Masina_(disambiguation)" class="mw-redirect mw-disambig" title="Masina (disambiguation)">Masina</a>, <a href="/wiki/Masini_(disambiguation)" class="mw-redirect mw-disambig" title="Masini (disambiguation)">Masini</a>, <a href="/wiki/Masino" title="Masino">Masino</a>, Masiol, <a href="/wiki/Maso_(disambiguation)" class="mw-redirect mw-disambig" title="Maso (disambiguation)">Maso</a>, <a href="/w/index.php?title=Masolini&amp;action=edit&amp;redlink=1" class="new" title="Masolini (page does not exist)">Masolini</a>, Masolo, <a href="/wiki/Masoni_(disambiguation)" class="mw-redirect mw-disambig" title="Masoni (disambiguation)">Masoni</a>, <a href="/wiki/Masotti" title="Masotti">Masotti</a>, Masselli, Massetti, <a href="/wiki/Massini" title="Massini">Massini</a>, <a href="/wiki/Masucci" title="Masucci">Masucci</a>, <a href="/wiki/Masuccio_(disambiguation)" class="mw-redirect" title="Masuccio (disambiguation)">Masuccio</a>, <a href="/wiki/Masulli" class="mw-redirect" title="Masulli">Masulli</a>, <a href="/wiki/Masullo" title="Masullo">Masullo</a>, Masutti, Masutto, and several others.<sup id="cite_ref-HouseOfNames_2-1" class="reference"><a href="#cite_note-HouseOfNames-2">&#91;2&#93;</a></sup>
</p>
